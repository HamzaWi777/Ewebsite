## 🚀 Getting Started (5 steps)

1. **Read** [QUICKSTART.md](./QUICKSTART.md)
2. **Install** Node.js and MySQL
3. **Create** database and `.env` files
4. **Run** `npm install` and `npm run dev`
5. **Visit** http://localhost:5173
